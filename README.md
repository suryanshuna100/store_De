# API-driven Data Pipeline (Spark + AWS)

## Overview
This project builds a data pipeline that ingests data from an API, processes it using Apache Spark, and stores it in AWS S3.

## Tech Stack
- Python
- Apache Spark
- AWS S3
- Athena (planned)
- Airflow (planned)

## Project Structure
- ingestion/: API scripts  
- spark/: transformations  
- airflow/: orchestration
